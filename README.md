MP3 Search

Sistema desenvolvido para realizar buscas e gerenciamento de arquivos MP3 de forma rápida e intuitiva, permitindo localizar músicas por nome, artista ou palavras-chave. O projeto visa transformar uma aplicação inicialmente executada via terminal em uma solução com interface gráfica amigável e organizada.

Projeto (API)

O MP3 Search surgiu da necessidade de facilitar a localização e organização de arquivos de áudio armazenados em computadores. A proposta é oferecer uma ferramenta simples, eficiente e acessível para usuários que desejam encontrar músicas rapidamente sem depender de plataformas externas.

O projeto segue a metodologia ágil SCRUM, permitindo entregas incrementais e melhoria contínua do produto ao longo das sprints.

Objetivo do Projeto

Desenvolver uma aplicação capaz de realizar buscas inteligentes em bibliotecas de arquivos MP3, permitindo ao usuário localizar músicas de maneira rápida e organizada.

Objetivos Específicos

- Facilitar a busca por músicas armazenadas localmente;
- Organizar informações dos arquivos encontrados;
- Melhorar a experiência do usuário através de uma interface intuitiva;
- Aplicar conceitos de programação, versionamento e metodologias ágeis;
- Evoluir um protótipo de console para uma aplicação completa.
